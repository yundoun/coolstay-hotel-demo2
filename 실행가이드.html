경주 황리단길 팰리스 호텔
예약 데모 사이트 — 실행 가이드

처음 실행하기

Windows

  1. 받으신 폴더를 열어주세요
  2. setup 파일을 찾아서 두 번 클릭하세요 (파일 이름이 setup.bat으로 보일 수도 있습니다)
  3. 검은색 창이 열리면서 자동으로 설치가 진행됩니다. 창을 닫지 마세요
  4. 설치가 끝나면 인터넷 브라우저(크롬 등)가 자동으로 열립니다
파란색 "Windows의 PC 보호" 경고가 나오면 → 추가 정보 → 실행 클릭
파일 이름 뒤에 .bat이 안 보이는 경우: 폴더 상단 보기 → 파일 확장명 체크하면 보입니다

Mac

  1. 받으신 폴더를 열어주세요
  2. setup 파일을 찾아서 두 번 클릭하세요 (파일 이름이 setup.command로 보일 수도 있습니다)
  3. 터미널 창이 열리면서 자동으로 설치가 진행됩니다. 창을 닫지 마세요
  4. 설치가 끝나면 인터넷 브라우저(사파리 등)가 자동으로 열립니다
"확인되지 않은 개발자" 경고가 나오면: 화면 좌측 상단 사과 아이콘 → 시스템 설정 → 개인정보 보호 및 보안 → 하단의 확인 없이 열기 클릭

사이트 설정하기

브라우저가 열리면 관리자 페이지가 나타납니다.

여기서 호텔 정보를 입력할 수 있습니다:

  • 호텔 연결 (API 키) — 쿨스테이에서 받은 모텔키를 입력하면 객실과 가격이 자동으로 연동됩니다
  • 기본 정보 — 호텔 이름, 주소, 전화번호
  • 배너 이미지 — 메인 화면에 보이는 큰 사진 (최대 5장, 최소 1장 필수)
  • 소개 이미지 — 호텔 소개 영역에 보이는 사진 (최대 5장, 최소 1장 필수)
  • 인사말 / 소개 — 호텔을 소개하는 글
  • 찾아오는 길 — 주차 안내, 주변 관광지 정보

내용을 수정한 뒤 화면 아래 저장 버튼을 눌러주세요.

사이트 미리보기

관리자 페이지 오른쪽 위의 사이트 미리보기를 클릭하면 실제 손님에게 보이는 화면을 확인할 수 있습니다.

종료하기

검은색 창(또는 터미널 창)을 닫으면 서버가 종료됩니다.

다시 보고 싶으면 setup 파일을 다시 두 번 클릭하세요.

소스가 업데이트된 경우

  1. SVN에서 업데이트 받으세요
  2. setup 파일을 다시 두 번 클릭하세요

변경된 내용이 자동으로 반영됩니다.

문제가 생겼을 때

"Node.js가 설치되어 있지 않습니다" 라고 나오는 경우

보통 자동으로 설치됩니다. 자동 설치가 안 되면 아래 순서를 따라해 주세요:

  1. 인터넷 브라우저에서 https://nodejs.org/ko 접속
  2. 화면의 초록색 버튼(LTS라고 적힌 것) 클릭 → 파일 다운로드
  3. 다운로드된 파일 두 번 클릭 → 다음 → 다음 → 설치 (모두 기본값)
  4. 설치 끝나면 컴퓨터를 재시작
  5. 다시 setup 파일을 두 번 클릭

브라우저가 열렸는데 화면이 안 나오는 경우

  • 검은색 창이 아직 열려있는지 확인하세요 (닫으면 서버가 꺼집니다)
  • 5~10초 기다린 뒤 브라우저에서 새로고침 해보세요
    Windows: 키보드 F5 / Mac: 키보드 ⌘ + R

설정을 저장했는데 반영이 안 되는 경우

브라우저에서 새로고침 해보세요 (Windows: F5 / Mac: ⌘ + R)

위 방법으로도 해결이 안 되면

검은색 창에 나온 글자를 화면 캡처해서 담당 개발자에게 보내주세요.

  • Windows: 키보드 Print Screen 키
  • Mac: 키보드 ⌘ + Shift + 3
