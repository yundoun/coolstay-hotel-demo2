# 소노호텔앤리조트 레퍼런스 분석

> 사전 분석 완료본. Planner는 이 파일을 읽고 계약서를 작성한다.
> 소노 사이트를 직접 크롤링하지 말 것.
> 스크린샷은 `harness/reference/screenshots/` 하위에 폴더별로 정리되어 있다.

## 사이트 개요

- URL: https://www.sonohotelsresorts.com
- 프레임워크: Nuxt.js (Vue 3) + Pinia
- 디자인 톤: 럭셔리 미니멀, 다크톤 GNB, 풀스크린 히어로, 넉넉한 여백

### 예약 위젯 상세

> 📷 `roomFilter/roomFilter-1,2,3.png`

```
┌──────────────────────────────────────────────────────────────────────────┐
│ 📍 지역 또는 숙소  │ ⚙ │  04.24 금요일  →  04.25 토요일  │ 객실 1  인원 1 ∨ │ [Search] │
│                   │   │      1박                        │               │          │
└──────────────────────────────────────────────────────────────────────────┘
```

**하단 고정 (sticky bottom)** — 페이지마다 하단에 있는 페이지는 모두 하단 고정.

**호텔 검색 (roomFilter-1):**
- 텍스트 입력 시 **실시간 자동완성** 드롭다운
- 트리 구조: `소노펫 비발디파크 [펫]` → `비발디파크 [리조트, 호텔]` → `· 소노벨 A` / `· 소노벨 B·C` / `· 소노캄` / `· 소노벨 D (호텔)`
- 각 항목에 아이콘: 🏠 리조트/호텔 객실 안내, 📊 가이드맵
- `델피노 [리조트, 호텔]` 등 다른 지역도 표시

**날짜 선택 (roomFilter-2):**
- 2개월 동시 표시 캘린더 팝오버
- 체크인 날짜 클릭 → 체크아웃 날짜 클릭으로 범위 선택
- 오늘 이전 날짜는 비활성 (회색)

**객실/인원 선택 (roomFilter-3):**
- 객실: `[- 1 +]` 카운터
- 대인: `[- 1 +]` 카운터
- 소인: `[- 0 +]` 카운터
- 하단 안내: 리조트 기준인원 / 호텔 기준인원 설명
- `[적용]` 버튼 (갈색)
